# Vim Tutorial

## Modes
1. Insert : `i`
2. Visual : `esc`
3. Command Line 

## Quit
`:q`

## Quit and Discard
`:q!`

## Write + Quit
`:wq`

## Write
`:w`

## Navigation
- H : Left
- J : Down 
- K : Up
- L : Right

# Delete
`X`

## Delete a Line
`dd`

## Undo
`u`

## Paste
`+p`

## Run Command
`:!<command>`
